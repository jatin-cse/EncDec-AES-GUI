# EncDec-AES-GUI

A simple Java GUI application for AES encryption and decryption of text files using a password.

## Features

- AES-GCM encryption with password-derived 256-bit key
- Encrypt text and save to a file
- Decrypt files and show filename, encrypted, and decrypted content
- Drag-and-drop support for decryption
- Show/hide password toggle
- Copy output to clipboard

## Usage

1. Compile and run:

```bash
javac EncDec.java
java EncDec
````

2. Enter text and password, then click **Encrypt & Save**.
3. Enter password and click **Decrypt File** to decrypt a file.
4. Drag and drop files to decrypt quickly.
5. Use the **Show Password** checkbox to view or hide the password.

## Requirements

* Java 8 or higher
